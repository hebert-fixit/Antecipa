# Security Policy 🔒

## Supported Versions

We actively support and patch the latest version of **Antecipa** on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

---

## Reporting a Vulnerability

If you discover a security vulnerability in this simulator, please do **not** open a public issue. Instead, report it privately to the maintainer:

* **Contact Email**: (Vulnerability report contact)
* **Response Time**: We aim to review and respond to all reports within 48 hours.

Thank you for helping keep Antecipa secure! 💎
